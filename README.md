4. 核心功能说明

- 4.1 导航与页面切换
  - 侧边导航 + 顶部入口
  - Hash 路由切换页面（无需额外路由库）
    
- 4.2 人生K线图（重点功能）
  - 使用 ECharts 折线图
  - X轴：年份；Y轴：运势（0-100）
  - 动态从左到右缓慢绘制，默认 1x 下完整时长 30000ms
  - 增强交互：暂停/继续、重播、倍速切换（0.5x/1x/1.5x/2x）、进度显示
  - 加载失败有错误提示与重试机制
    
- 4.3 排盘输入（复杂表单）
  - 日历体系、公历/农历、闰月、时区、时辰、年月日时分联动
  - 省市联动输入、性别、时刻精度、真太阳时开关
  - 咨询主题/行业/婚恋/学历/子女规划/健康关注等业务字段
  - 输入摘要实时预览
    
- 4.4 业务流程
  - 登录/注册/找回密码流程
  - 命理师筛选与主页查看
  - 订单结算、支付成功、个人中心记录联动
    
- 4.5 空态与错误态标准化
  - 统一空数据展示组件
  - 图表加载失败统一告警样式 + 操作入口

5. 视觉与规范要求落实
- 暗色星空背景 + 玻璃拟态卡片风格
